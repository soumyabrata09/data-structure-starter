## This starter Package is housing the Data structure concepts
### This package can be added a dependency to build files

**Pre-Requisite**
 - Maven in local
 - JDK 1.8 minimum
 
 ````
 mvn clean compile
 ````
 And then
 ````
 mvn clean install
 ````
